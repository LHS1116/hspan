# hspan-front

> A vue project for hspan

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## Routes
``` bash

# 登录/注册
/login

# 注销
/logout

# 文件列表
/my/files

#收藏列表
/my/favorites

#分享列表
/my/shares

#个人信息
/my/me

#文件详情
/file/:id

#用户信息
/user/:id

```
